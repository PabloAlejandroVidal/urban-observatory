# Urban Observatory

Plataforma fullstack para visualización y análisis de datos urbanos y geoespaciales.

## Stack previsto

- Frontend: Angular
- Backend: NestJS
- Database: PostgreSQL + PostGIS
- Auth: JWT + roles
- Maps: Leaflet o MapLibre GL JS
- Deploy: VPS, Nginx, Docker

## Estructura

- `apps/api`: backend NestJS
- `apps/web`: frontend Angular
- `docs`: documentación técnica
- `infra`: configuración de despliegue
- `data`: datasets de ejemplo o referencias a fuentes públicas
